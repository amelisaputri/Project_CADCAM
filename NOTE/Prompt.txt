🖥️ Apa itu Server dan Client?
✅ Client
Client itu adalah program yang digunakan oleh pengguna (user) untuk mengirim dan menerima pesan.
Contohnya: Aplikasi chat yang dijalankan oleh setiap karyawan di komputernya.
Tugas client:
•	Login ke server
•	Kirim pesan ke server
•	Terima pesan dari server
•	Tampilkan pesan di layar
✅ Server
Server adalah program pusat yang menerima semua koneksi dari client dan bertugas meneruskan pesan ke orang yang dituju.
Tugas server:
•	Terima koneksi dari banyak client
•	Simpan siapa saja yang sedang online
•	Terima pesan dari si pengirim
•	Kirim ulang pesan itu ke si penerima
________________________________________
📶 Gimana Cara Kerjanya?
1.	Client login → kirim info ke server: LOGIN|123456|Budi
2.	Server simpan: "Oh, si Budi (123456) sudah online"
3.	Budi kirim pesan ke Andi → CHAT|123456|654321|Halo, Andi!
4.	Server terima → cari client Andi → kirim pesan itu ke Andi
5.	Andi terima pesan di aplikasi chat-nya

🎯 Analogi Singkat
Bayangkan kamu kirim surat via kantor pos:
•	Client = Pengirim surat
•	Server = Kantor Pos
•	Penerima = Client lain
Kamu (client) kirim surat ke kantor pos (server).
Kantor pos cari alamat penerima (client lain), dan kirimkan suratnya.
Semua lewat jalur pusat (server), jadi tidak langsung antar pribadi.
